# Desafío 2 - Layout de E-commerce con Grid 🛒

Este proyecto consiste en la maquetación de una tienda online básica para validar el uso de estructuras complejas y responsivas combinando **CSS Grid** y **Flexbox**.

## 🛠️ Requerimientos Implementados

- **Layout General (CSS Grid):** Estructura modular construida mediante la propiedad `grid-template-areas` definiendo zonas claras para `Navbar`, `Aside Bar` (Sidebar), `Grilla de productos` y `Pie de página`.
- **Grilla de Productos (CSS Grid):** Despliegue de tarjetas de productos dinámicas y adaptables usando `repeat(auto-fit, minmax(250px, 1fr))`.
- **Barra Lateral (Flexbox):** Distribución y ordenamiento vertical de las categorías utilizando Flexbox.
- **Tarjetas de Productos (Flexbox):** Estructuración interna de cada tarjeta (imagen, textos de información y botón de compra) alineados y distribuidos verticalmente con Flexbox de manera uniforme.

## 🧰 Tecnologías Utilizadas

- **HTML5:** Marcado semántico y limpio.
- **CSS3:** Maquetación avanzada combinando **Grid Layout** y **Flexbox**.
- **Git & GitHub:** Control de versiones y despliegue del sitio.
